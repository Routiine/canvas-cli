# Canvas CLI Assets

## Logo
The Canvas CLI logo should be saved as `canvas-cli-logo.png` in this directory.

The logo features:
- A deep blue/purple gradient background
- Interconnected grid of glowing nodes
- Central bright burst point
- Neural network-like pattern representing AI intelligence

## Logo Usage
- Main README header
- Documentation
- CLI splash screens
- Marketing materials

To add the logo image:
1. Save the logo image as `canvas-cli-logo.png` in this directory
2. The README will automatically display it at the top
3. The logo represents Canvas CLI's intelligent, interconnected nature
