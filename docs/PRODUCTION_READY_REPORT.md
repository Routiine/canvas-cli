## 🎬 Getting Started

### Installation
```bash
npm install -g canvas-cli
```

### First Run
```bash
# Initialize project
canvas init

# Run comprehensive analysis
canvas /sentient analyze

# Optimize codebase
canvas /sentient optimize

# Prepare for deployment
canvas /sentient ship
```

### Daily Workflow
```bash
# Morning check
canvas /sentient monitor

# Before coding
canvas /sentient analyze

# After features
canvas /sentient optimize

# Before deployment
canvas /sentient ship
```
